# Maumee Dashboard

Internal dashboard project for field-level crop, weather, and historical data exploration.

## Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/sg459/maumee-dashboard.git
   cd maumee-dashboard
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Data setup**

   This project requires a large geospatial data file that is not stored in GitHub due to file size limits.

   - Copy `OH_regrow_fieldID_geometry.parquet` from your shared drive (or the wherever it's stored e.g., Teams) to:
     ```
     data/raw_local/OH_regrow_fieldID_geometry.parquet
     ```

4. **Run the dashboard**
   ```bash
   streamlit run app.py
   ```

## Project Structure

- `app.py` - Main Streamlit dashboard application
- `data/` - Data directory
  - `metadata/` - Variable definitions and crop codes
  - `processed/` - Processed datasets (generated by scripts)
  - `raw_local/` - Local data files (not tracked in git)
- `scripts/` - Data processing scripts
- `docs/` - Documentation

## Requirements

See `requirements.txt` for Python dependencies.
